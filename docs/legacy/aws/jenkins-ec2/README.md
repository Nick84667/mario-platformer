The active target architecture has moved to:

- Hetzner Cloud
- Jenkins VM
- k3s Kubernetes VM
- ArgoCD
- SonarQube VM
- GHCR container registry

This folder is kept for historical reference only and should not be used for the current GitOps architecture.
