# Security Policy

## Supported Versions

This repository is maintained as part of a DevSecOps and GitOps lab.

## Reporting a Vulnerability

Please open a private security advisory or contact the repository owner.

## Security Controls

The CI/CD pipeline is designed to include:

- TypeScript checks
- ESLint checks
- SonarQube analysis
- Trivy filesystem scan
- Trivy container image scan
- GitOps-based deployment through ArgoCD
